Allow to set pattern lock for wp admin screens.

== Description ==

Pattern Lock allows to set pattern lock for wp admin screens. You can set pattern lock for your screen according to your choice.

== Installation ==

1. Upload the `pattern-lock` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.

== Frequently asked questions ==

## How to use
1. First Activate Plugin.
2. Save Pattern Settings from Pattern Lock Settings Menu.
3. Set Pattern Lock of your choice.
4. Click on Lock Screen red button on top bar.
5. Then Pattern Lock screen will appear.
